## VERSIONE CORTA DEL PROMPT

Crea una web app marketplace chiamata **Dental Clean**, dedicata ai prodotti odontoiatrici. Deve funzionare come un marketplace professionale verticale: i produttori si registrano, pagano un abbonamento mensile e possono pubblicare prodotti singolarmente o tramite import massivo da file Excel. Il cliente entra nel sito, usa una grande barra di ricerca centrale, trova prodotti, filtra risultati, confronta prezzi, legge recensioni e acquista. La homepage deve mostrare prodotti in evidenza, bestseller, offerte, nuovi arrivi e categorie principali. I produttori possono pagare extra per mettere i prodotti in evidenza con piani mensili o trimestrali.

Progetta homepage, risultati ricerca, scheda prodotto, carrello, checkout, dashboard venditore, dashboard admin, pagina pricing venditori, upload Excel, gestione prodotti, recensioni e area cliente.

Lo stile deve essere medicale, pulito, moderno, professionale, affidabile, mobile responsive, con colori bianco, azzurro medicale, blu profondo e verde acqua. Crea design system completo, componenti UI, flussi UX cliente/venditore/admin e layout pronti per sviluppo.
